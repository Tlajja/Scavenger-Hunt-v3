# Scavenger-Hunt
PS inzinerija 1

## About
Photo scavenger hunt. I.e. a specific task for what kind of photo to capture is provided and people have a limited amount of time to upload such photo. Photo with the most votes wins.

Project was made using C#, dotnet, react, signalR, Cloudinary, etc.

## Goals (UPDATE NEEDED?)
What the project should have when finished:
1. Photo function - store & upload
2. Generating task(s)
3. User writing and submitting their own tasks
4. Authentication - Create user profile
5. Voting on photos
6. Comments/chatting/reacting
7. Leaderboard
8. Timers and/or notifications - game and voting times.
9. User activity - how many people are using the website at the moment
10. Create/join challenge - private or public
